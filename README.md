# Learn Enough HTML, CSS, and Layout to be Dangerous

This is a sample website created by following *Learn Enough HTML, CSS, and Layout to be Dangerous*, a practical tutorial on front-end web technologies. By following this tutorial, you'll learn enough to be *dangerous*, meaning enough HTML and CSS to efficiently solve problems, and to design, build, and deploy modern, professional-grade websites.

For more information about the Learn Enough series of tutorials, visit the [official website](https://www.learnenough.com/).
