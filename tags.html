Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones. For more information on all of the HTML elements and their tags, visit the MDN Web Docs.

Block Elements

Tag Name Purpose
div division generic container for flow content
h1–h6 headings include a heading (levels 1–6)
header header represents a group of introductory content
li list item include an item on a list
ol ordered list include a numbered list of items
p paragraph include a paragraph of text
table table include a table
td table data include a table data cell
th table heading include a heading for a table column
tr table row include a row of data in a table
ul unordered list Include a non-numbered list of items

Inline Elements

Tag Name Purpose Example Result
a anchor link to other resources <a href="https://www.learnenough.com/">Learn Enough</a> Learn Enough
code code add monospaced text, like source code <code>source code</code> source code
em emphasised make emphasised text <em>technical sophistication</em> technical sophistication
img image include an image <img src="https://bit.ly/1MZAFuQ alt="Michael Hartl">
span span generic container for inline content <span>Call me Ishmael.</span> Call me Ishmael.
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
